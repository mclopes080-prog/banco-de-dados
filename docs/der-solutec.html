DER - Loja Virtual Solutec

Modelo inicial para venda de cameras, alarmes e produtos de seguranca eletronica.

CLIENTE

  • PK id_cliente
  • nome
  • email
  • telefone
  • cpf_cnpj
  • endereco

CATEGORIA

  • PK id_categoria
  • nome
  • descricao
  • status

PRODUTO

  • PK id_produto
  • FK id_categoria
  • nome
  • descricao
  • preco
  • estoque
  • imagem

PEDIDO

  • PK id_pedido
  • FK id_cliente
  • data_pedido
  • status
  • valor_total

ITEM_PEDIDO

  • PK id_item
  • FK id_pedido
  • FK id_produto
  • quantidade
  • preco_unitario
  • subtotal

FORNECEDOR

  • PK id_fornecedor
  • nome_fantasia
  • cnpj
  • telefone
  • email

Relacionamentos

  • CATEGORIA 1:N PRODUTO - uma categoria possui varios produtos.
  • CLIENTE 1:N PEDIDO - um cliente pode realizar varios pedidos.
  • PEDIDO 1:N ITEM_PEDIDO - um pedido possui um ou mais itens.
  • PRODUTO 1:N ITEM_PEDIDO - um produto pode aparecer em varios itens de pedido.
  • FORNECEDOR N:N PRODUTO - pode ser implementado com a tabela PRODUTO_FORNECEDOR em sprint futura.
