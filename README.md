# Solutec - Sprint inicial

Loja virtual academica para venda de cameras, alarmes e solucoes de seguranca eletronica.

## Como executar

Com XAMPP/PHP instalado:

```txt
C:\xampp\php\php.exe -S 127.0.0.1:8000 -t "C:\Users\User\OneDrive\Desktop\PROJETO-BANCO DE DADOS"
```

Depois acesse:

```txt
http://127.0.0.1:8000/index.php
```

## Entregaveis da sprint

- Layout PHP da loja: `index.php`
- Template PHP: `includes/template.php`
- Dados dinamicos de produtos: `data/products.php`
- CSS e JS: `assets/css/style.css` e `assets/js/script.js`
- DER em PDF: `docs/der-solutec.pdf`
- DER editavel/imprimivel: `docs/der-solutec.html`
- Orientacao para evidencia DNS local: `docs/dns-local.md`
- Exemplo de linha do hosts: `docs/hosts-example.txt`
